## 🛠 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Flutter 3.x (Dart) |
| 视频内核 | media_kit + libmpv (硬件解码) |
| 音频 |多音源插件 |
| 数据存储 | sqflite (SQLite) + SharedPreferences |

---

## 📋 功能

- **影视** — 多源聚合，拼音搜索，分类浏览
- **电视** — M3U/TXT 直播源，频道分组，EPG 节目指南
- **音乐** — 在线播放，歌单/排行榜/专辑/歌手/MV，歌词滚动，均衡器，定时关闭，播放速度调节
- **资源库** — 影视源/电视源/音源管理，TVBOX 订阅
- **系统** — 明暗主题，语言切换，缓存清理，更新检测
